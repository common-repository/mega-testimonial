== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Your Plugin is activated and Now goto settings to tune your Testimonial content appreance.
4. You will get different shortcode to activate Mega Testimonials activity. (Ex. Default Shortcode -> [gb_testimonial])
5. Read the shortcode docs in the settings panel and the further use.
6. Enjoy !
